# DER

Coloque nesta pasta o arquivo exportado do Astah.

Arquivo atual:

```text
der_hft_simulator.jpeg
```

Nomes tambem aceitos caso o DER seja reexportado:

```text
der_hft_simulator.pdf
der_hft_simulator.png
```

No Astah, exporte usando uma destas opcoes:

```text
File > Print > Microsoft Print to PDF
```

ou capture a tela com:

```text
Win + Shift + S
```

Depois salve a imagem como `der_hft_simulator.png`.
